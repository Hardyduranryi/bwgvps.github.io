实用 - CentOS/Debian/Ubuntu快速一键安装纯净版系统

不论我们选择哪个商家的VPS主机，商家一般会自带系统镜像安装的，而且我们网友也较多的会选择商家提供的镜像。当然商家提供的镜像一般也都没有太大的问题，不过有些时候确实镜像存在一些问题，我们有些功能是无法安装的。我们会选择自定义镜像，不过有些商家不提供自定义镜像服务，那有没有其他办法自定义镜像呢？在这篇文章中，我们提供网上整理的可以快速安装纯净版CentOS、Debian、Ubuntu系统。

搬瓦工中文网在调试一些软件的时候，默认自带系统会有安装软件，我们甚至无法确定哪些软件存在，所以直接用纯净版系统还是比较好的，这样缺少什么软件我们就再去安装。我们看看如何进行安装纯净版系统。

Tip： 在这篇文章中提供的脚本安装纯净版系统，CentOS支持6.X，不支持以上版本。

第一、商家自带镜像系统登入

这里我们需要先安装商家自带的镜像，不管我们安装什么都可以的。因为后面我们会重新安装纯净版镜像，所以不影响。具体也可以参考《 选择正确选择BandwagonHost VPS主机系统和内核》，唯独我们需要注意的是重新安装搬瓦工VPS主机系统之前需要先STOP关闭当前系统，然后才可以安装新系统，不关闭是不可以新安装的。完毕之后，我们获取到新系统的ROOT端口和密码，然后我们登录SSH，重新安装之后需要等待大约一分钟才可以完毕，不可以立即登入。

第二、安装系统需要的软件支持

在这篇文章中，我个人比较喜欢Debian，所以我需要去安装Debian系统纯净版。在这篇文章中，我们就安装这个系统。

apt-get update -y 

    apt-get install -y xz-utils openssl gawk file

第三、快速安装Debian8 纯净版系统

在这篇文章中，搬瓦工中文网就只安装Debian8纯净版系统，直接一键脚本。

bash <(wget --no-check-certificate -qO- 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh') -d 8 -v 64 -a

登入SSH，丢上脚本，回车之后我们可以看到自动安装。

我们看到这样的界面，不要管，不要以为是错误了，其实等待一会会安装成功的。稍微等几分钟即可。

第四、登录新的纯净版系统

新系统默认端口是22，root密码是 Vicer，我们登录即可。

这样，我们可以看到已经安装完毕纯净版Debian8系统。
