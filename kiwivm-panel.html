第一、如何登入kiwivm面板

如果我们当前搬瓦工VPS主机中拥有已经购买的机器，可以看到上图，点击kiwivm control panel按钮，然后进入面板管理。其实我们也可以登入独立的面板管理的，不同的机器可以有单独的入口。这个我们后面在介绍，在这篇文章中，我们还是以kiwivm面板为主。

第二、kiwivm 主面板 Main Controls介绍

直接通过上图，我们可以看到主面板的信息。我们可以看到当前服务器的机房所处位置、服务器IP地址、端口、当前服务器运行状态，如果我们以后需要重启、STOP停止当前机器，以及KILL清理进程都是可以用到。以及我们可以看到当前服务器的内存、硬盘、流量占用情况，包括我们当前已经安装的系统。

第三、kiwivm 监控系统 Detailed statistics

我们可以从Detailed statistics中看到当前搬瓦工VPS主机的监控信息，比如CPU、硬盘读写、网络信息情况，这个一般的主机商都有的，不过我们很多人都不关心。或者我们也可以使用第三方监控服务器的性能情况，毕竟我们也不能24小时打开面板，可以使用跟踪工具，后面我们在试试其他第三方工具。

第四、kiwivm WEB端shell工具

我们可以看到搬瓦工VPS主机kiwivm面板中有Root shell - basic、Root shell - advanced、Root shell - interactive三个工具，可以实现WEB端SSH(shell)执行的，一般我们用的并不多。一般我们在创建完毕系统之后，都会使用shell这类ssh工具进行客户端管理。后面我们再单独用详细的文章分析这三个工具到底分别做什么用途的。其实一般的商家只有一个工具就可以，这里居然放三个。

第五、kiwivm 重新安装系统Install new OS

重新安装系统是必须要用的功能。我们可以随意、随时的进行重新安装搬瓦工VPS系统。我有在 基础 - 搬瓦工VPS重新安装Linux系统过程及注意事项文章中有介绍到重新安装系统的方法。我们要记住，重新安装系统之前需要先在主面板中STOP当前的机器状态，然后才可以重新安装。重新安装系统后系统密码会变动一个随机密码，以及数据全部被格式化。如果有重要数据，需要提前备份。

第六、kiwivm 二次密码验证

估计我们大部分站长使用的比较少，其实在有些主机商或者海外域名注册商是有这些功能的，提供需要Two-factor authentication二次密码保护，比如用手机激活码的方式进行登录账户验证。在搬瓦工VPS主机中，登录面板的时候需要验证，一般我们不设置，其实主机面板也很少会丢失账户吧，我们设置账户密码复杂一些。

第七、kiwivm 重置服务器ROOT密码

这个功能我们也常用到的，比如当前服务器的ROOT密码忘记或者希望重新设置，可以在SSH中进行设置，也可以在这里进行设置一个随机密码。但是实际上设置的密码不够复杂，我们最好还是通过SSH进行设置复杂的密码。

第八、kiwivm 重置kiwivm密码

如果我们kiwivm面板是单独管理这台机器的，那我们可以设置这台搬瓦工VPS主机的独立面板管理。比如我们这台机器要交给其他人管理，不至于把所有的账户给他，可以单独给一台机器给他们管理。所以，只要给这台服务器的IP地址，以及当前设定的kiwivm面板密码就可以。

第九、kiwivm 迁移数据中心

搬瓦工VPS与其他主机商不同的是，商家可以迁移到其他机房，而且IP地址是变动的。这个也算是搬瓦工VPS主机的特点之一。如果我们有需要迁移机房的，可以参考使用。需要注意的是，需要注意的是，迁移机房之后IP地址是要变动的，但是数据还是在的。在 搬瓦工VPS自主任意更换机房实现换IP文章中，我们有看到如何切换机房。有时候我还是建议切换之前先备份数据。

第十、kiwivm 快照和自动备份

搬瓦工VPS主机有提供免费的快照备份和自定定时备份服务，一般的主机商没有提供或者是需要收费服务的。但是在这里是免费的。我们在kiwivm面板中可以看到有Snapshots、backups选项，看到上图。快照备份是需要手动备份的，但是我们可以设置一个永久备份快照，其他都只能备份30天。对于自动备份，是根据实际数据情况，自动备份的。
